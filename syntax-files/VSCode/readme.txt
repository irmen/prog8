A contributed VSCode extension providing syntax highlighting and more, can be obtained from:

  https://marketplace.visualstudio.com/items?itemName=sadlogic.prog8-lang



DEPRECATED VERSION:

  https://github.com/akubiczek/Prog8-TmLanguage-VsCode
  
Copy the Prog8 folder and its content into the ./extensions/ Visual Studio Code folder, Located at:

- Windows: \%USERPROFILE%\.vscode\extensions
- MacOS: ~/.vscode/extensions
- Linux: ~/.vscode/extensions
